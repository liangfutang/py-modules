# 版本环境管理
## 版本管理(pyenv)
```shell
# 安装指定版本
pyenv install 3.12.2
# 查看当前管理的所有版本
pyenv versions
# 切换版本
pyenv global 3.12.2
```

## 环境管理(pipenv)
```shell

# 查看当前项目的虚拟环境
pipenv --venv
# 为项目安装虚拟环境
pipenv install
```
