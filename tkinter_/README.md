
# pyinstaller
## pyinstaller命令参数说明
--onefile：将所有文件打包成一个单独的可执行文件。
--name：指定生成的可执行文件的名称。
--icon：指定要使用的图标文件的路径。
--noconfirm：在生成可执行文件时自动继续，不显示确认消息。
--debug：显示调试信息。
--strip：去除符号表信息，减小可执行文件大小。
--upx：使用UPX压缩可执行文件（需要安装UPX工具）。
--clean：在生成之前，清理PyInstaller缓存并删除临时文件。
